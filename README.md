# Stepper-control-with-knob
Stepper control with knob was developed for precise control of 'Angle of Attack' of an airfoil in a wind-tunnel for aerodynamic studies.  
This was made under the request of Prof. Paradha Saradhi, BITS Pilani Hyderabad Campus.
